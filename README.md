# kmpatch
### KMPlayer's advertisement is annoying!
##### So we remove them :P


Usage:

`kmpatch.py [options] file1 file2 ...`

Example:

`kmpatch.py -au KMPlayer.exe`

Options:

 - `--no-ads` or `-a`
     - Remove annoying sidebar advertisements
 - `--no-update` or `-u`
     - Remove update checking
